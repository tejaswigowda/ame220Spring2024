    Examples:
    ```
    #d6a6b9 = rgb(214,166,185) 
    d6 = d * 16^1 + 6 * 16^0 = 13 * 16 + 6 * 1 = 214
    a6 = a * 16^1 + 6 * 16^0 = 10 * 16 + 6 * 1 = 166
    b9 = b * 16^1 + 9 * 16^0 = 11 * 16 + 9 * 1 = 185
    ```

    ```
    rgb(123,56,245) = #7b38f5 
    123 = 16 * 7 + 1 * 11 = 7b 
    56 = 16 * 3 + 1 * 8 = 38 
    245 = 16 * 15 + 1 * 5 = f5
    ```
